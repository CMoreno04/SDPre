What is the difference between ‘switch’ and ‘if/else’? When would you use one versus the other?

The difference is:

  • "Switch" will switch the users input with another variable declared (double, int or String).
  • On the other hand, "If / Else" will have some parameters. If the the parameter is accomplished then run first option, else consider this other parameter and run.

What is ‘void’ in a method declaration? How does this differ from declaring a method with a return type?

  • The "void" declaration tell us that the method has no specific return, while with a declared method (double, int or string), the return has to be in accordance with he declaration.
